# Patricio Bunt - Developer Toolkit

Collection of developer tools and utilities. PWA that works offline.

Live at [patriciobunt.com](https://patriciobunt.com)

## Features

- PWA with offline support
- Dark/light themes, customizable accent colors
- Favorites system
- Resizable sidebar
- Service worker caching
- Fullscreen mode for tools
- Customizable default landing page
- Toggle visibility of upcoming tools

## Tech

HTML, CSS, vanilla JavaScript. Service workers for PWA stuff. No frameworks.

## Setup

Static site, serve it however you want. For local dev:

1. Clone the repo
2. Run a local server (python -m http.server or npx serve)
3. Open in browser

PWA features need HTTPS (or localhost).
